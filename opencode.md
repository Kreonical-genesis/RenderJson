## Детальное описание задач

### 2. Статистика использования (🟡)

**Описание**: Отслеживание общего количества рендеров, моделей, времени работы.
**Файлы**: index.html
**Stack**: localStorage

### 3. Предпросмотр анимации (🟡)

**Описание**: Слайдер для пролистывания кадров анимации в панели камеры.
**Компоненты**:

- Панель с слайдером
- Кнопки воспроизведения/остановки
- Определение типа анимации (sequence/texture)

### 4. Множественные источники света (🟡)

**Описание**: Панель управления освещением с пресетами (studio, dramatic, minecraft).
**Компоненты**:

- Ambient light control
- Directional lights (до 3)
- Point lights (до 2)
- Пресеты освещения

### 5. Режим "Скетч" (🟡)

**Описание**: Шейдер для эффекта sketched/outline.
**Stack**: Three.js EffectComposer + ShaderPass

### 6. Режим "Коллаж" (🟡)

**Описание**: Создание одного PNG из всех выбранных моделей сеткой.
**UI**: Кнопка в панели рендера, настройки колонок и размера ячеек.

### 7. Пакетный экспорт (🟠)

**Описание**: Возможность выбрать формат и разрешение для каждой модели.
**UI**: Модальное окно с настройками, создание ZIP.

### 8. Поддержка .bbmodel (🔴)

**Описание**: Парсер .bbmodel файлов Blockbench.
**Stack**: JSON парсинг + Three.js геометрия

### 9. Скины игроков (🟠)

**Описание**: Загрузка скина по нику через Mineskin API.
**UI**: Панель с полем ввода ника.

### 10. Интеграция с облаком (🟠)

**Описание**: Автозагрузка рендеров в Imgur или Discord webhook.
**Stack**: Fetch API

### 11. Виртуальный скроллинг (🔴)

**Описание**: Рендер только видимых элементов в списке (для 1000+ моделей).
**Stack**: Virtual scrolling algorithm

### 12. Виртуальный пол и тени (🔴)

**Описание**: Плоскость под моделью с мягкими тенями.
**Stack**: Three.js shadows + PlaneGeometry

### 13. Поддержка .jem моделей (🟠)

**Описание**: Парсинг Jade Eclipse Models - формат из Minecraft 1.8+.
**Stack**: JSON парсинг + Three.js геометрия

### 14. Загрузка .jar архивов (🟠)

**Описание**: Поддержка Minecraft JAR файлов для извлечения моделей.
**Stack**: JSZip + фильтрация контента

## 15. ПОДДЕРЖКА .JEM МОДЕЛЕЙ (JADE ECLIPSE MODELS)

**Категория: 🟠 СЛОЖНЫЕ**

### Описание

Парсинг .jem файлов - формата Jade Eclipse Models (Minecraft 1.8+).

### Структура .jem

```json
{
  "model": {
    "name": "model_name",
    "texture": "texture_path",
    "elements": [
      {
        "from": [x, y, z],
        "to": [x, y, z],
        "faces": {
          "north": { "uv": [u1, v1, u2, v2], "texture": "#texture" },
          ...
        }
      }
    ],
    "animations": [...]
  }
}
```

### Особенности

- Поддержка встроенных текстур (base64)
- Анимации и трансформации
- Наследование от других моделей

---

## 16. ЗАГРУЗКА .JAR АРХИВОВ

**Категория: 🟠 СЛОЖНЫЕ**

### Описание

Поддержка загрузки Minecraft JAR файлов (client.jar, server.jar) для извлечения моделей.

### Особенности

- Распаковка JAR (это ZIP архив)
- Извлечение моделей из assets/minecraft/models/
- Извлечение текстур из assets/minecraft/textures/
- Фильтрация только нужных моделей (items, blocks)
- Кэширование распакованных данных

### UI

- Добавить .jar в список поддерживаемых форматов
- Индикатор распаковки JAR
- Выбор версии JAR (1.8, 1.12, 1.16, etc)
