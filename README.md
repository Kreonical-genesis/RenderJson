# RenderJson

Веб-инструмент для рендеринга 3D-моделей Minecraft из ресурспаков в форматы PNG и GIF.

## Возможности

- **Загрузка ресурспаков** — Drag & Drop ZIP/JAR файлы напрямую в браузер
- **Поддержка моделей** — Блочные модели (`item/handheld`, `item/generated`) и модели с элементами
- **Анимации** — Автоматическое объединение моделей в анимации (ходьба, атака, и т.д.)
- **Режимы рендера**
  - PNG — статический кадр
  - GIF — вращение модели или покадровая анимация
- **Настройка камеры** — Позиция, угол обзора, зум, пресеты
- **Пакетная обработка** — Рендер сразу нескольких моделей
- **Экспорт** — ZIP архив или скачивание по одному файлу
- **Превью** — Предпросмотр моделей в списке

## Использование

1. Перетащите ZIP или JAR файл ресурспака в область загрузки
2. Нажмите «Загрузить модели»
3. Выберите нужные модели из списка
4. Настройте параметры рендера (режим, разрешение)
5. Нажмите «Начать Рендер»

### Настройки камеры

Панель инструментов позволяет настроить:

- **X, Y, Z** — позиция камеры
- **Zoom** — масштаб
- **Пресеты** — готовые настройки (Default, Isometric, Top Down, Front, Side)

### Горячие клавиши

- `Ctrl+A` — выбрать все модели
- `Ctrl+D` — сбросить выбор
- `Ctrl+R` — начать рендер

## Технологии

- [Three.js](https://threejs.org/) — 3D рендеринг
- [JSZip](https://stuk.github.io/jszip/) — работа с ZIP архивами
- [gif.js](https://jnordberg.github.io/gif.js/) — создание GIF анимаций
